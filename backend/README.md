## Car Auction BACKEND

Stack: 
- Typescript
- NestJS
- TypeORM
- PostgreSQL
- Swagger 

local URLs
http://localhost:3001/

http://localhost:3001/api - Swagger

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
