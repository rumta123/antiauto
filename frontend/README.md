## Car auction Frontend

What we use:
- <b>Next.js</b>
- <b>FSD</b> architecture
- <b>Orval</b> for generate API from Swagger specs
- <b>React-hook form</b>
- <b>TailwindCSS</b>
- <b>Gravity-UI kit</b>


Orval
download OpenAPI specification 
<code>npm run api:download</code>
generate API
<code>npm run api:generate</code>
